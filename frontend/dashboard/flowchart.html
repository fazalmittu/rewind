How Rewind Works

Vision-based workflow recording for browser applications

1
Record
Chrome extension captures everything
Clicks
Button presses, link clicks, selections
Inputs
Text typed into fields and forms
Screenshots
Visual state after each action
Metadata
URLs, timestamps, element info
2
Process
AI pipeline extracts structure
STAGE 1
Canonicalize
Group similar screens into types
STAGE 2
Segment
Identify distinct workflow instances
STAGE 3
Synthesize
Generate templates with parameters
